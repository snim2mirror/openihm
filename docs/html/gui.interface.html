executeMultipleResultsQueries(queries)¶

This method is idential to self.executeResultsQuery except that it takes a list of query strings, executes each in turn and returns a corresponding list of results.

executeMultipleUpdateQueries(queries)¶

This method is idential to self.executeUpdateQuery except that it takes a list of query strings and executes each in turn

executeResultsQuery(query)¶

Execute a query for which the database will return results. For example a SELECT query.

executeUpdateQuery(query)¶

Execute a query that needs to be committed to the database. For example, an INSERT or UPDATE query.
